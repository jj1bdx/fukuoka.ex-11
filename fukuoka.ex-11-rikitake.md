# Code BEAM STO とは

* Erlang User Conference が元の名前
* 1994年に初めて開催（1996年を除く）
* 1999年以降のproceedingsが残っている
* 2009年以降はErlang Solutionsが開催

---

# なぜ Code BEAM?

* ブランドの付け替え
* Erlangだけではなくなった
* Elixirの台頭
* LFE, efene, Alpaca, clojerl, luerl, erlog
* →BEAMという名前への統合
* （BEAMが浸透しているかどうかは微妙）
